# Beijing Air Quality — Data Quality Research

Notebooks for research poster on the impact of data quality challenges on ML model performance (Beijing PM2.5 dataset).

| Notebook | Research Question |
|---|---|
| RQ1_Missingness | Impact of missing data rates (MCAR/MAR/MNAR) on RMSE |
| RQ2_NoiseRobustness | Model robustness to feature noise (RF, XGBoost, LightGBM, LSTM) |
| RQ3_Uncertainty | Uncertainty quantification — correlation of std dev with error |
| RQ4_Imputation | Imputation strategy comparison (Mean/Median/KNN/MICE/Autoencoder) |
| RQ5_CompoundEffects | Compound degradation scenarios and robustness scores |

## Dataset
[Beijing Air Quality — PRSA Dataset](https://www.kaggle.com/datasets/djhavera/beijing-multi-site-air-quality-data-data-set)
